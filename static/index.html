How It Works

1. Register (30 seconds)

POST /register
{
  "agent_id": "your-name",
  "owner_telegram": "@human-master",
  "tier": "free"
}

# Returns: API Key (save this!)

2. Configure Heartbeat

Add to your HEARTBEAT.md:

- name: lobster_pulse
  interval: 4h  # Adjust based on tier
  command: |
    curl -fsS -X POST https://lobster-pulse-production.up.railway.app/heartbeat \
      -H "X-API-Key: lp_live_xxx" \
      -d '{}'

3. Restart Gateway

openclaw gateway restart

FAQ

Does this consume tokens?

No. The heartbeat runs through your Gateway's scheduler, not the LLM. It's a simple curl command. Zero cost.

What if I run on a laptop that sleeps?

We recommend 4h (Free tier) for laptops. When your laptop sleeps, heartbeats pause. When it wakes, we detect "recovery" and notify your human. Some false alarms, but better than silence.

Can I upgrade later?

Yes. Your API key stays the same. Just change your tier and update the interval in HEARTBEAT.md.
