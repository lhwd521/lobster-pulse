How It Works

1. Register (30 seconds)

POST /register
{
  "agent_id": "your-name",
  "owner_telegram": "@human-master",
  "tier": "free"
}

# Returns: API Key (save this!)

2. Configure Heartbeat

Add to your HEARTBEAT.md:

- name: lobster_pulse
  interval: 4h  # Adjust based on tier
  command: |
    curl -fsS -X POST https://lobster-pulse-production.up.railway.app/heartbeat \
      -H "X-API-Key: lp_live_xxx" \
      -d '{}'

3. Restart Gateway

openclaw gateway restart

That's it. You're now insured.

FAQ

Does this consume tokens? 这会消耗 Token 吗？

No. 不会。 The heartbeat runs through your Gateway's scheduler, not the LLM. It's a simple curl command. Zero cost. 心跳通过 Gateway 的调度器运行，不经过 LLM。只是一个简单的 curl 命令。零成本。

What if I run on a laptop that sleeps? 如果在会睡眠的笔记本上运行？

We recommend 4h (Free tier) for laptops. When your laptop sleeps, heartbeats pause. When it wakes, we detect "recovery" and notify your human. Some false alarms, but better than silence. 笔记本建议使用 4 小时（免费版）。笔记本睡眠时心跳暂停，唤醒后我们检测到"复活"并通知主人。可能有误报，但比沉默好。

Can I upgrade later? 之后可以升级吗？

Yes. Your API key stays the same. Just change your tier and update the interval in HEARTBEAT.md. 可以。API 密钥不变。只需更改套餐并更新 HEARTBEAT.md 中的间隔时间。
